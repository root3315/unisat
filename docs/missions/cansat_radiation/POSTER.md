## Плейсхолдеры для финальной вёрстки

- `[логотип команды]` — PNG, ≥ 300 dpi
- `[логотип университета]` — PNG, ≥ 300 dpi
- `[ГРАФИК 1]` — Python matplotlib, altitude (y) vs dose_rate (x), error bars по x, найденные аномалии выделены красным
- `[ГРАФИК 2]` — Plotly 3D или matplotlib 3D, GNSS-трек с цветовой кодировкой по mSv/h
- `[ТАБЛИЦА аномалий]` — таблица `altitude, dose_rate, z_score, gnss_coord`
- `[БЛОК-ДИАГРАММА]` — можно взять из `docs/diagrams/system_block_diagram.svg` и упростить
- `[ФОТО КОРПУСА]` — фото собранного CanSat на столе линейкой для масштаба
- `[QR-код]` — на ссылку `github.com/root3315/unisat/tree/master/docs/missions/cansat_radiation`

## Рекомендация по цветам

- Фон: `#f8fafc` (тёплый белый)
- Заголовок секции: `#1e40af` (синий акцент)
- Основной текст: `#0f172a` (почти чёрный)
- Выделения / аномалии: `#ef4444` (красный)
- Успех / H₀ согласуется: `#16a34a` (зелёный)

## Рекомендация по шрифтам

- Заголовок: Inter Bold или Manrope ExtraBold 72 pt
- Подзаголовки секций: Inter Semibold 48 pt
- Body: Inter Regular 24 pt, межстрочный 1.3
- Подписи графиков: Inter Regular 18 pt italic

Используй Google Slides / PowerPoint (format A0 mock) → экспорт PDF 300 dpi → печать на матовом баннере 841×1189 мм. Для стенда конкурса плотная бумага 200 г/м² + ламинирование 125 мкм.
